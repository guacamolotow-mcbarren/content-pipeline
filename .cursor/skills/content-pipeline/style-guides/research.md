# Style Guide: Исследование

## Формат
- Аналитический отчёт / исследование рынка или технологии
- Структура близка к white paper
- Длина: 20000–80000 знаков (зависит от глубины)

## Тон
- Академически-деловой, нейтральный
- Автор — экспертная группа «Инфосистемы Джет»
- Факты, данные, методология; минимум субъективных оценок

## Структура

```markdown
# [Название исследования]

## Аннотация
[2–3 абзаца: цель, метод, ключевые выводы]

## 1. Введение
[Кontext, постановка проблемы, цели исследования]

## 2. Методология
[Как собирали данные, выборка, ограничения]

## 3. [Тематический блок 1]
[Анализ, таблицы, графики]

## 4. [Тематический блок 2]
...

## N. Выводы и рекомендации
[Структурированные выводы, практические рекомендации]

## Приложения
[Таблицы данных, глоссарий, источники]
```

## Обязательные элементы
- Титульный блок: название, дата, автор(ы), компания
- Аннотация (executive summary) в начале
- Нумерованные разделы
- Таблицы с данными (Markdown или описание для вставки)
- Ссылки на источники
- Глоссарий терминов (если >10 специальных терминов)
- Блок «О компании» в конце

## Визуализация
- Таблицы Markdown для сравнительных данных
- Описание графиков: «[График 1: динамика рынка X, 2020–2025]»
- Инфографика — текстовое описание для дизайнера

## Лексика
- Безличные конструкции: «исследование показало», «по данным опроса»
- Цифры с указанием источника
- Аббревиатуры — в глоссарии

## Запреты
- Неподтверждённые утверждения без пометки «по оценкам»
- Рекламные блоки внутри исследования
- Копирование чужих исследований без цитирования
